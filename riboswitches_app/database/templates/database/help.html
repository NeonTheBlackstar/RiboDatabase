About the RISE

RISE (RIboswitch databaSE) is a database which provides information about riboswitches.

There are several ways to retrieve the information. You can use:

  • Quicksearch on main page
  • Main search
  • Taxonomy browsing

Search

To find individual riboswitch record, you can use quicksearch or the main search.

Quicksearch

  • Start typing either a riboswtich name (eg. [..]) or a ligand (eg. alanine).
  • If you don't remember any of this names, you can use displayed suggestions.

Main search

  • The search module displays all the riboswitches from the database.
  • The information is in a tabular format.
  • You can easily filter the results using inputs which are on the bottom of the page.



Browse

If you'd like to browse riboswitches instead of search for a specific one, you should use browse.

Taxonomy browser

  • You can select either the single species or some of any intermediate level.
  • The taxonomy is based on NCBI Taxonomy Database

Ligand browser

  • The next option for browsing the database is a ligand browser.
  • This browser displays all of the ligands from database.


Record

The last step of browsing the database is a record page

  • This page contains all the information for specific riboswitch
  • It is displayed in tabular form
