What is RISE?


Riboswitches are fragments of mRNA, which affect on a control of transcription,
translation, splicing and RNA stability.

RISE is a web app for the management of riboswitch database

There are several ways to retrieve the information. You can use:

  • Quicksearch on main page
  • Main search
  • Taxonomy browsing
